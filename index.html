Distributed Hash Decryption

  • Java | TCP/IP

One-way hashing algorithms are practically infeasible to invert, demanding a timely brute-force attack to obtain a single [key, hash] pair. A table of computed [key, hash] pairs for that algorithm may be used to drastically speed up the process, but these tables are just as slow to generate, and costly to store due to their size.

I developed an application to decrypt NTLM-encoded passwords by generating such a table while distributing the work and storage. A master-slave architecture was used to distribute work in blocks across a set of remote TCP connected worker nodes.

Artificial Animals

  • Unity Game Engine | Aseprite | C#

A small-scale, competitive, multi-agent simulation of the interaction between carnivorous predators (wolves), herbivorous prey (deer), and their environment. Intro to A.I. final project displaying how agents use limited perception to act within it's best interests.

Prompted by hunger and thirst, deers attempt to graze flowering plants while moving in packs to evade wolves. A fear response balanced by stamina is used to run away from threats. Constant reproduction and predation keeps the environment dynamic.
